Adhiyan 3.0.1 Released

11 March, 2008

The next minor bug fix version of Adhiyan Mozilla extension has been released on 17th February 2008.

You can download and install Adhiyan 3.0.1 as specified at http://adhiyan.mozdev.org/installation.html.

Bugs fixed in this version:

  • Fixed minor bug in English detection rules

This release is also available from Mozilla Extensions website. The browsers with previous version of Adhiyan installed already should get updated automatically.

I would like to thank Sethu from Thamizha! Free Tamil Computing group for his contribution in the testing of this extension.

Please use http://adhiyan.mozdev.org/bugs.html if you would like to raise any bug or feature request.

PS: Due to some other priorities, The release news has been delayed.
