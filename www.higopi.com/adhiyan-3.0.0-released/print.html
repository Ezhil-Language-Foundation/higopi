Adhiyan 3.0.0 Released

15 January, 2008

The latest version of Adhiyan Mozilla extension has been released on 15th January 2008.

You can download and install Adhiyan 3.0.0 as specified at http://adhiyan.mozdev.org/installation.html.

Bugs fixed in this version:

  • Encoding should be skipped for English content
  • Canonical equivalence for Ogaram, Augaram related bug
New Features in this version:

Pre-selected website URLs

You can enter the URLs and required encoding conversion for the websites that you browse regularly. When you browse next time, the conversion will be triggered automatically. We have also included set of known websites and related conversions

New conversions

Added support to DailyThanthi, Dinamani and related websites

Right click context menu changes

Now you can select which conversions to be shown in the right click context menu. If automatic conversion is selected, it will show the required conversion also even if its selected to be hidden from the right click context menu.

Soon this release will be available from Mozilla Extensions website. When available, the browsers with previous version of Adhiyan installed already will get updated automatically.
