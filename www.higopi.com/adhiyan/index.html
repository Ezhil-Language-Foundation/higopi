Adhiyan Firefox and Thunderbird Extension

When you are browsing through a TSCII/TUNE/TAB/TAM website you may not be able to read the content without installing the required fonts. Adhiyan firefox extension is designed to solve this issue.

What is Adhiyan?

Adhiyan is a Firefox/Thunderbird extension using which you can convert and read TSCII/TUNE/TAB/TAM encoded websites and emails to Unicode. This way you can read the TSCII/TUNE/TAB/TAM content without installing the required fonts.

Adhiyan is useful especially in the computers where the user is not given with the permission to install new fonts.

Adhiyan supports webpages which requires login and web pages that uses cookies. It also converts the web pages which uses FRAMES.

How to Install Adhiyan on Firefox?

  1. Visit Adhiyan web page at Mozilla Addons and follow the instructions specified there.

Alternatively, Just follow the simple steps mentioned below.

  1. Click here to download Adhiyan extension and save it to your local disk.
  2. Launch Firefox
  3. Open the directory where you saved the downloaded the adhiyan-3.0.1-fx+tb.xpi file
  4. Drag the adhiyan-3.0.1-fx+tb.xpi file and drop it in to Firefox window
  5. This will open the Extension manager
  6. Click on the "Install Now" button and restart Firefox after installation is done
  7. That's all... your firefox is now enabled with Adhiyan extension.

How to Install Adhiyan on Thunderbird?

  1. Visit Adhiyan web page at Mozilla Addons and follow the instructions specified there.

Alternatively, Just follow the simple steps mentioned below.

  1. Click here to download Adhiyan extension and save it to your local disk.
  2. Launch Thunderbird
  3. Click on Tools -> Extensions from the menu
  4. Open the directory where you saved the downloaded the adhiyan-3.0.1-fx+tb.xpi file
  5. Drag the adhiyan-3.0.1-fx+tb.xpi file and drop it in to Extensions window of Thunderbird
  6. This will open the Extension manager
  7. Click on the "Install Now" button and restart Thunderbird after installation is done
  8. That's all... your Thunderbird mail client is now enabled with Adhiyan extension.

What if I am using Internet Explorer?

Right now this extension is supported only in Firefox and Thunderbird. I am working on to make it available for Internet Explorer and other browsers. Please check back this web page for updates about the availability of this extension for Internet Explorer.
