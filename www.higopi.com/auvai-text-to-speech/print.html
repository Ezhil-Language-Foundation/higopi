Auvai Text to Speech

05 October, 2007

Auvai Text to Speech is a Java API and Swing based application for Tamil Unicode Text to Speech Conversion.

The code is developed entirely using Java 1.4 core APIs and JLayer external library (supplied along with the Installer). Only requirement to run this utility is JRE 1.4 or above.

The code uses Speech Reproduction technology.

The application is platform/os independent.

The API can be used in any Java application to enable Tamil Unicode Text to Speech.

This code is currently under development. Release of the initial version is expected by the end of January 2012.
