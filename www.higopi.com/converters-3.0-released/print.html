Converters 3.0 Released

03 October, 2008

The next version of transliteration converters has been released.

New features in this release.

  • Tamil 99 Keyboard layout added to Thagadoor Tamil Converter
  • Online keypad map help which pops up when you type. if you feel it annoying., you can switch it off using converter options in the left side menu
  • Key pad map will pop up in the converter window itself instead of a different window as it was. This can also be switched on/off
